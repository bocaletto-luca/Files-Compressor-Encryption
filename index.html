Files Compressor with Encryption

Compress your files and secure them with encryption in one simple Python app.

Description

This tool lets you compress, encrypt, and extract archives in a single workflow:

  • Compress: Uses zlib at highest compression to shrink your files.
  • Encrypt: Secures compressed data with Fernet symmetric encryption.
  • Extract: Decrypts and decompresses archives back to original files.

Screenshots

Archive Extension Management

By default archives use the .7bl extension (e.g. data.7bl), but you can choose any extension or custom filename—you'll still get encrypted, compressed content.

User Interface

A simple GUI built with tkinter offers:

  • File selection dialogs for source files and target folders
  • Input field for your encryption key
  • Buttons to Compress & Encrypt or to Extract & Decrypt
  • Status label for real-time feedback and error messages

Installation

# Clone the repository
git clone https://github.com/bocaletto-luca/Files-Compressor-Encryption.git
cd Files-Compressor-Encryption

# Install dependencies
pip install cryptography

Usage

  1. Launch the app: 
    python main.py
  2. To compress & encrypt:
    • Select source file(s)
    • Enter an encryption key
    • Click Compress & Encrypt
  3. To extract & decrypt:
    • Select archive file (.7bl or your custom extension)
    • Enter the same encryption key
    • Click Extract & Decrypt

Technical Stack

  • tkinter – GUI toolkit
  • zlib – Data compression
  • cryptography.Fernet – Symmetric encryption
  • os, sys – File and path handling
